uint32_t OPAMP_InitTypeDef::InvertingInput

Specifies the inverting input in Standalone & PGA modes

  • In Standalone mode: i.e. when mode is OPAMP_STANDALONE_MODE & PGA mode: i.e. when mode is OPAMP_PGA_MODE This parameter must be a value of OPAMP Inverting Input
  • In Follower mode i.e. when mode is OPAMP_FOLLOWER_MODE This parameter is Not Applicable
